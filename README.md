# Slimefun4 中文魔改版

主打的是极简(更快)

相比于中文版，做了如下改动:
- 移除了部分无用代码
- 添加了部分中文版没有的功能
- 抢先修复部分bug
- 优化部分代码
- 移除了1.16与1.17的支持
- 新增物品类型：可加速的机器(消耗电的机器，如充电台那些，简介中带有可加速即可)
- 移除了对原生Spigot服务端的支持，仅支持Paper及其下游服务端
